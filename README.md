Knowbox Data Analyst Level 1 Exam
================

胡润研究院每个季度都会发布《胡润大中华区独角兽指数》，针对估值超过 10 亿美金的企业进行排名。这份练习将利用此数据探索国内当前的互联网格局。

数据已经过清洗与处理，储存于 `raw.csv`，包含了以下维度,

    1. ranking # 排名
    2. ranking_chg # 排名变化
    3. name_zh # 企业名称
    4. name_en # 企业英文名称
    5. valuation # 估值
    6. key_person # 掌门人
    7. headquarter # 总部
    8. industry # 行业
    9. investors # 部分投资机构

-----
